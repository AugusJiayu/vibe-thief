设计系统：Open

这是一个以内容为核心、极度克制的系统。通过近乎单色的调色板和经典的衬线字体，它将所有注意力都引导到信息本身。

核心概念

极度克制的单色体系

整个系统建立在黑色和浅灰的二元对比上。功能色彩被有意省略，以维持视觉的绝对宁静。

经典的衬线体与清晰的层级

选择Times New Roman作为全站字体，唤起一种永恒的、专注于文字内容的阅读感。

规律与呼吸的间距

所有间距都基于4px的基础单元，确保布局的节奏感和可预测性。组件内部紧凑，组件之间宽松。

低调的层级深度

极少使用阴影，卡片与背景平级或仅通过极淡的1px边框来定义。层级关系主要依靠背景色差异和间距。

按钮组件

按钮有三种状态：默认、主要（用于关键操作）、次要。禁用状态应显著降低对比度。

输入框组件

输入框在悬停时边框颜色加深，聚焦时显示强化边框和外轮廓。

代码示例

/* 创建一个主按钮 */
.btn-primary {
  background-color: #1a1a1a;
  color: #ffffff;
  border: 1px solid transparent;
  border-radius: 4px;
  height: 36px;
  padding: 0 12px;
  font-weight: 500;
  transition: all 150ms ease-out;
}
/* 创建一个卡片 */
.card {
  background-color: #ffffff;
  border: 1px solid rgba(0,0,0,0.1);
  border-radius: 8px;
  padding: 24px;
}
